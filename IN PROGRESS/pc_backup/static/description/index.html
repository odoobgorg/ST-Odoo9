Description

Backs up important files on your desktop computers
This software is incomplete and lacks the core functionality to do backups
Process
1. User logins
2. User goes to "/backup/client/download" and downloads client
3. User installs backup client
4. The backup client is preloaded with server address and login creds to simplify setup
5. Once installed the users picks the directories they want backup up (default documents,downloads and desktop)
6. The backup automatically occurs once a day uploading the files to the server
7. The user can restore individual files at any time by using a link inside the client

Future Features
  • Backup client which automatically backups computer once a day
  • User selects folders to backup
  • Incremental backup
  • Store all versions of a file for rollback
  • View files in backup before restore
  • Restore individual files from backup
  • Daily, weekly and monthly backups
  • Automatic removal of old backups
